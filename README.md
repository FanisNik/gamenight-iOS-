##  Features

- 3 games in 1 app 
- 2 difficulty modes: Standard & Extreme
- Skip questions with random sip notifications
- Clean, intuitive UI for easy navigation



##  Built With
- [Swift](https://developer.apple.com/swift/)  
- [SwiftUI](https://developer.apple.com/xcode/swiftui/)  
- [PhotosUI](https://developer.apple.com/documentation/photosui)

##  Platform
iOS 15 and above

##  Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/GameNight-iOS.git
   ```
2. Open GameNight.xcodeproj in Xcode.
3. Build and run on a simulator or physical device (iOS 15+ recommended).
